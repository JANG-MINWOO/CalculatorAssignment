# 프로젝트 구성
1. Lv 1 클래스 없는 계산기 만들기
2. Lv 2 클래스를 활용한 계산기 만들기
3. Lv 3 제네릭, enum을 활용한 계산기(lambda&stream 활용x)

## 사용법
1. 파일을 다운받은 후, 콘솔이 실행 가능한 환경에서 실행한다.
2. 프로그램의 안내에 따라 숫자와 연산자를 입력한다.
3. 결과를 확인한 뒤 원하는 옵션을 선택한다.
